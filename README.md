# 📌 EduTrack – Project Management Case Study

## 🚀 Project Description
EduTrack is a student productivity web application project focused on helping users manage daily study tasks and track progress. This project demonstrates end-to-end project management using Agile methodology.

---

## 🎯 Objective
- Plan and manage a real-world project using Agile practices  
- Break down requirements into structured tasks  
- Execute and track progress using sprints  

---

## 🛠️ Tools Used
- Jira (Project Management & Sprint Tracking)  
- Microsoft Excel (Planning & WBS)  

---

## 🧠 Project Approach

1. Defined project scope using Project Charter  
2. Created BRD & FRD for requirements  
3. Broke down work into:
   - Epics  
   - User Stories  
   - Subtasks  
4. Planned and executed 3 Sprints  
5. Managed tasks using Jira Board  
6. Identified and mitigated risks  

---

## 📅 Sprint Plan

- **Sprint 1:** User Authentication + Task Management  
- **Sprint 2:** Progress Tracking + Enhancements  
- **Sprint 3:** Testing + Finalization  

---

## 📊 Key Features

- User Signup & Login  
- Task Management (Add, Edit, View Tasks)  
- Dashboard (Daily Task Overview)  
- Progress Tracking (Daily/Weekly)
